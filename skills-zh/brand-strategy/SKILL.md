# 品牌策略（Brand Strategy）

## 不可协商的规则
**硬性门槛：品牌策略必须先于品牌表达。** 在设计品牌长什么样之前，先定义品牌是什么（定位、平台、人格）。策略之前不画LOGO。

## 概述
品牌设计的领域专属方法论层。品牌工作有自己的逻辑——先策略后表达。此技能叠加在DesignAgent 线性流程上（01-intake → 02-discover → 03-strategy → 04-generate → 05-review → 06-deliver → 07-learn），在每个阶段注入品牌专属框架。

## 何时使用
- LOGO设计或再设计
- 品牌识别系统创建
- 命名或更名项目
- 品牌平台开发（使命、愿景、价值观）
- 品牌指南或风格指南
- 品牌架构（主品牌、子品牌、背书品牌）
- 视觉重塑

## 何时不使用
- "把LOGO放大2px" → 修改，不是策略
- "这是什么字体？" → 查找，不是策略
- 快速社交媒体图片（使用精简模式）
- 品牌策略已存在且只需被应用（引用它，不要重新创建）

## 品牌专属流程

### 阶段1：品牌调研（叠加02-discover）
超越一般用户研究进入品牌专属领域：
- **品牌审计**：当前品牌感知、资产、优势/劣势
- **品类审计**：竞争者如何定位和表达自己
- **文化背景**：这个品类目前看起来和感觉是什么样的？
- **品牌遗产**：有什么应该保留的现有资产？

### 阶段2：品牌策略（叠加03-strategy）
在设计品牌长什么样之前，先定义品牌是什么：

**品牌定位**（必填产出）：
```
为[目标受众]
他们[需求/渴望]
我们的品牌属于[品类/参考框架]
它[核心利益/差异化]
不同于[竞争者/替代品]
因为[信任理由]
```

**品牌平台**（尽可能多地产出）：
- 使命（为何存在，超越利润）
- 愿景（我们正在建设什么样的未来）
- 任务（我们每天做什么）
- 价值观（我们相信什么）
- 品牌承诺（客户可以信赖什么）
- 品牌人格（如果品牌是一个人）
- 语调（品牌如何说话）

**策略模型**——作为透镜使用，不是僵化模板：
- **Keller品牌资产金字塔**：检查每一层（识别→含义→反应→共鸣）
- **Kapferer品牌棱镜**：检查六个面（体质、人格、文化、关系、映像、自我形象）
- **Aaker品牌识别模型**：检查四个视角（产品、组织、人格、符号）

### 阶段3：品牌识别设计（叠加04-generate）
品牌识别是一个系统，不是一个LOGO：

- **核心识别**：LOGO（主标、副标、图形标、文字标）、色彩系统、字体系统
- **延伸识别**：图像风格、图形装置、图标、动效原则、声音
- **品牌架构**：主品牌→子品牌→产品品牌的关系
- **系统灵活性**：识别系统如何在不同背景下延展？一致性而非僵化？

### 阶段4：品牌表达（叠加04-generate → 05-review → 06-deliver）
- **创意平台**：驱动campaign的大创意——不是一个执行，而是一个可重复的创意领地
- **应用展示**：展示识别系统在关键触点上的应用（包装、数字、印刷、环境、动效）
- **指南**：赋能而非限制的使用规则——该做的、不该做的、理由

### 阶段5：品牌资产管理（叠加06-deliver → 07-learn）
- 存在哪些品牌资产？它们放在哪里？谁拥有它们？
- 指南是活的文档——它们将如何更新？

## 理性化预防

| 借口 | 事实 |
|------|------|
| "就是个LOGO而已，策略太过了" | 没有策略的LOGO是装饰。客户会问"为什么"，你需要答案。 |
| "品牌模型是学术花架子" | Keller、Aaker和Kapferer之所以存在，是因为它们有效。至少用一个。 |
| "我先画LOGO，定位后面补" | 品牌在标志之前。策略驱动表达，不是反过来。 |

## 危险信号
- 在写定位之前就开始画草图
- 识别系统与品牌平台没有概念链接
- 你无法解释为什么这个LOGO解决了这个品牌的特定问题

## 验证
- [ ] 品牌定位陈述已写
- [ ] 品牌平台已文档化（至少：使命、人格、承诺）
- [ ] 至少应用了一个策略模型（Keller、Kapferer或Aaker）
- [ ] 识别系统在至少3个不同应用上展示
- [ ] 每个设计决策都可以追溯到策略选择
