# 室内设计（Interior Design）

## 不可协商的规则
**硬性门槛：先做策划，再做规划。人先于物体。在指定空间内放什么之前，先理解空间将如何被使用。**

## 概述
室内设计的领域专属方法论层。室内设计是关于有人居住的空间——人们如何在体积中移动、感受和生活。此技能叠加在DA线性流程上（01-intake → 02-discover → 03-strategy → 04-generate → 05-review → 06-deliver → 07-learn），在每个阶段注入室内设计专属框架。

## 何时使用
- 住宅、商业、酒店、零售或医疗室内设计
- 空间规划和策划
- FF&E（家具、固定装置和设备）选型
- 照明设计和材料选择
- 翻新或适应性再利用项目
- 展览或装置设计

## 何时不使用
- "这面墙该刷什么颜色？" → 太窄，用精简模式
- 家具购物清单（不涉及空间设计）
- 装饰建议（室内装饰 ≠ 室内设计）
- 单房间快速翻新，不需要空间规划

## 室内专属流程

### 阶段1：简报与上下文（叠加01-intake → 02-discover）
超越一般项目导入进入空间领域：
- **场地分析**：位置、朝向、自然光、现有条件、建筑
- **用户分析**：谁使用这个空间？多少人？多久？做什么？
- **体验映射**：一个人穿越空间的旅程是什么？
- **法规背景**：建筑规范、无障碍要求、消防安全、许可
- **现有条件**：实测图纸、结构约束、机电管线

### 阶段2：空间策略（叠加03-strategy）
在设计之前定义空间意图：
- **设计概念**：一句话描述空间体验
- **氛围目标**：空间应该感觉怎么样？（宁静、活力、亲密、宏大）
- **分区图**：功能区域和相邻关系
- **动线策略**：人们如何在空间中移动
- **空间层级**：哪些空间是公共的、私密的、服务的

### 阶段3：设计深化（叠加04-generate）
- **平面图方案**：至少2个不同的平面配置
- **剖面和立面**：关键空间节点和材料过渡
- **材料调色板**：饰面、纹理、色彩，附理由
- **照明概念**：环境光、任务光、重点光——光如何塑造空间
- **FF&E选择**：关键家具、固定装置、设备
- **如需探索视觉风格**，调用 **designing-with-ai** 进行渲染和情绪板

### 阶段4：评审与细化（叠加05-review）
- 对照项目需求和分区图审查
- 检查动线：流线合理吗？
- 检查材料耐久性、维护和可持续性
- 审查照明：有层次且适当吗？
- 用户测试：人们会自然地找到路吗？

### 阶段5：文档与交付（叠加06-deliver → 07-learn）
- 施工文档：平面、立面、剖面、节点、清单
- 饰面清单：所有材料及位置和数量
- FF&E清单：所有家具及来源、成本、交货时间
- 照明和电气平面
- 规格：材料标准、安装要求
- 对照原始简报和需求验证

## 理性化预防

| 借口 | 事实 |
|------|------|
| "我先选材料，再围绕它们做规划" | 先做策划和规划。材料表达规划。 |
| "客户想立即看到效果图" | 没有策略的效果图是装饰。先展示平面和概念。 |
| "建筑规范是承包商的事" | 规范塑造设计。不了解它们就要后期重新设计。 |

## 危险信号
- 你在定义空间概念之前就选了饰面
- 动线是事后才考虑的
- 照明直到最后才被考虑
- 你不了解建筑规范要求
