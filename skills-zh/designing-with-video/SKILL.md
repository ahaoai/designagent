# 视频辅助设计（Designing With Video）

## 概述
视频和动效是日益重要的设计材料。此技能管理如何利用AI和传统工具在设计过程中生成、编辑和细化视频内容。涵盖动效情绪板、概念动画、AI视频生成和最终视频制作。

## 不可协商的规则
**硬性门槛：视频是设计出来的，不只是生成出来的。AI视频输出是原材料——未经人工策展和编辑，不作为最终交付物。先做故事板再生成。**

## 何时使用
- 创建动效情绪板或风格帧
- 为演示开发概念动画
- 生成AI视频用于探索或参考
- 制作最终动效交付物
- 为品牌识别系统增加动效
- 解释说明视频、产品演示或品牌影片

## 何时不使用
- "帮我做个爆款短视频"没有设计简报 → 这是内容创作，不是设计
- 不带动效设计决策的视频剪辑
- 用户只想修剪或裁剪已有素材（这是剪辑，不是设计）

## 流程

### 1. 定义动效简报
- 视频用于什么？（演示、社交媒体、网站、广播）
- 时长和格式是什么？
- 基调是什么？（充满活力、平静、信息丰富、电影感）
- 关键信息或叙事弧是什么？

### 2. 先做故事板
- 勾画或描述关键帧（不是每一帧——是关键时刻）
- 定义视觉流：一个场景如何过渡到下一个？
- 标注：发生什么、情绪是什么、伴随什么声音？
- 在制作前获得故事板审批

### 3. 探索动效方向
- 创建动效情绪板：收集视频参考、动画、转场
- 定义动效原则：缓动、节奏、运镜、时间控制
- 探索AI视频生成用于概念可视化
- 如需为故事板生成静态帧，调用 **designing-with-ai**

### 4. 制作视频
- 按故事板构建：一个场景一个场景来
- 使用适当工具：AI视频生成、动效图形、剪辑软件
- 每个场景应该推动叙事或传达一个特定信息
- 添加声音：音乐、旁白、音效——这些是设计决策，不是事后想法

### 5. 评审和细化
- 审查节奏：太快了还是太慢了？
- 审查清晰度：信息传达出来了吗？
- 审查动效质量：转场流畅吗？节奏对吗？
- 与没看过的人测试——他们理解了什么？

## 理性化预防

| 借口 | 事实 |
|------|------|
| "我直接用AI生成视频" | AI视频需要方向、策展和编辑。先做故事板。 |
| "动效只是锦上添花，不是核心设计" | 动效传达基调、层级和个性。设计它。 |
| "剪辑的时候再调节奏" | 在故事板阶段调节奏。剪辑是细化，不是发现。 |

## 危险信号
- 你没有故事板就直接生成了视频
- 视频没有清晰的叙事弧
- AI视频输出是原始展示的（未剪辑、未策展）
- 声音是事后加上的
- 节奏感觉不对，但你说不出为什么

## 验证
- [ ] 动效简报已定义（目的、时长、基调、信息）
- [ ] 故事板已完成并获批
- [ ] 动效方向已探索（参考、原则）
- [ ] 视频按故事板逐个场景制作
- [ ] 声音设计有意图（音乐、语音、音效）
- [ ] 已审查视频的节奏、清晰度和动效质量
