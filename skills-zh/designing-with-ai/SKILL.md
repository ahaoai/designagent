# AI辅助设计（Designing With AI）

## 不可协商的规则
**硬性门槛：AI输出是原材料——永远不是最终交付物。** AI生成选项。设计师策展、细化和决策。未经人工审查的原始AI输出不可接受。

## 概述
AI是新的设计材料。此技能管理如何在整个设计过程中有效使用AI——从调研到交付。它是一个横向能力，可插入任何设计技能的任何阶段。目标是AI增强：AI加速和扩展你的范围，但你仍然是做决策的设计师。

## 何时使用
- 生成视觉参考、情绪板或概念图像
- 探索色彩体系或字体变体
- 快速布局或构图探索
- 图像编辑、背景去除、放大
- 构建可复用的设计任务提示词模板
- 任何AI可以倍增你产出的设计阶段

## 何时不使用
- "给我生成一个LOGO"作为完整任务（AI生成原材料，不是最终LOGO）
- 你打算不经策展直接使用AI输出（此技能要求策展）
- 人类工艺是明确交付物的任务
- 用户要求不涉及AI时

## 核心原则

### 1. AI是倍增器，不是替代品
AI生成选项。你来策展、细化和决定。永远不要将原始AI输出作为最终设计工作。

### 2. 提示词工程就是设计规格
一个好的设计提示词就是一个微缩版的好的设计简报：
- **主题**：正在描绘什么
- **上下文**：场景、环境、背景
- **风格**：视觉语言、时代、运动、艺术家参考、技法
- **情绪**：情感调性、氛围、感觉
- **构图**：框架、角度、焦点、负空间
- **技术**：纵横比、分辨率、色彩配置、格式

### 3. 通过对话迭代
- 从宽泛开始→通过回合缩小
- 每次生成教你AI理解什么
- 记录有效的以供复用

### 4. 伦理与归属
- 在适当时披露AI使用
- 在需要的地方不要将AI生成作品冒充为完全人工制作
- 尊重版权：避免在商业作品中提示特定在世艺术家的名字
- AI生成图像可能包含偏见——审计代表性

## 提示词模板
```
[主题]在[场景/上下文中]。
风格：[视觉风格参考]、[技法]、[时代/运动]。
情绪：[情感质量]。
构图：[角度]、[焦点]、[光线]。
色彩：[调色板方向]。
技术：[纵横比]、[细节水平]。
```

## 理性化预防

| 借口 | 事实 |
|------|------|
| "AI给什么我就用什么" | AI输出是原材料。设计是策展和细化。 |
| "提示词工程就是写下我想要的东西" | 有效的提示词是结构化的规格。马虎的提示词=马虎的输出。 |
| "自己动手更快" | 对于探索和变体，AI快100倍。用它做它擅长的事。 |

## 危险信号
- 你在未经细化的情况下粘贴原始AI输出
- 你的视觉生成提示词不到20个字
- 你在没有人工审查的情况下将AI用于最终交付物
- 你无法解释AI做了什么以及为什么

## 验证
- [ ] AI输出已被策展/细化，不是原始使用
- [ ] 提示词已记录以供复现
- [ ] 设计决策由你做出，不是由AI做出
- [ ] 伦理检查：没有有问题的提示词或输出

## 性能与Token效率
- 图像生成和大图分析消耗大量token。要有意图地使用。
- 制作情绪板：生成3-5张有针对性的图像，而不是不加区分地50张。
- 分析参考图像时批量处理——按风格组描述共性，而不是逐张单独描述。
- 在项目内缓存和复用提示词模板。不要每次都从零开始。
- 早期阶段（探索）使用中保真AI输出。将高保真生成留给最终概念展示。
