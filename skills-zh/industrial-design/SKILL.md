# 工业设计（Industrial Design）

## 不可协商的规则
**硬性门槛：用户和上下文研究必须先于形式生成。在设计产品形状之前，理解产品如何使用、在哪里使用以及如何制造。没有功能和上下文就没有形式。**

## 概述
工业设计的领域专属方法论层。工业设计是关于创造功能、可用、美观且可制造的实体产品。此技能叠加在DesignAgent 线性流程上（01-intake → 02-discover → 03-strategy → 04-generate → 05-review → 06-deliver → 07-learn），在每个阶段注入工业设计专属框架。

## 何时使用
- 消费产品设计
- 家具和照明设计
- 医疗或辅助设备设计
- 包装设计（实体）
- 硬件和电子产品设计
- 工具和设备设计
- 可持续产品开发

## 何时不使用
- "画个椅子" → 快速草图，用精简模式
- 平面包装设计（仅视觉，无实体产品）→ 使用 brand-strategy
- 工程分析或FEA（这是设计方法论，不是工程）
- 单零件修改，不需要用户上下文

## 工业设计专属流程

### 阶段1：调研和上下文（叠加01-intake → 02-discover）
- **用户研究**：产品如何使用？谁用？在什么情境下？
- **人体工学分析**：人体测量、握持、够取范围、姿势、交互力
- **竞争分析**：现有产品、专利布局、市场空白
- **制造研究**：可用工艺、材料、供应链
- **可持续性研究**：生命周期分析、材料影响、废弃处理
- **法规研究**：安全标准、认证、测试要求

### 阶段2：产品策略（叠加03-strategy）
在形式探索之前定义产品意图：
- **产品简报**：它解决什么问题？核心功能是什么？
- **使用场景**：用户如何一步步与它交互？
- **设计原则**：什么驱动设计？（耐用、简洁、可识别、可持续）
- **材料和工艺方向**：哪些材料和制造工艺在范围内？
- **目标成本和价格**：成本目标是多少？什么驱动成本？
- **可持续性目标**：回收含量、可回收性、可修复性、使用寿命

### 阶段3：形式开发（叠加04-generate）
- **草图探索**：跨2-3个方向至少10-20个概念草图
- **3D形式研究**：快速CAD或实体模型探索
- **人体工学测试**：检查比例、握持、可用性（用模型）
- **材料和饰面探索**：色彩、纹理、材料样品
- **如需探索视觉风格**，调用 **designing-with-ai** 进行渲染和可视化
- **设计迭代**：基于人体工学和功能反馈细化形式

### 阶段4：评审和测试（叠加05-review）
- **功能测试**：它按预期工作吗？
- **用户测试**：观察真实用户与原型交互
- **人体工学验证**：舒适度、易用性、安全性
- **制造审查**：在目标成本和质量水平下可行吗？
- **可持续性审计**：材料、包装、运输、废弃处理
- **法规审查**：它满足所需标准吗？

### 阶段5：工程和交付（叠加06-deliver → 07-learn）
- **工程文档**：CAD文件、技术图纸、公差
- **材料和饰面规格**：精确材料、色彩、纹理、表面处理
- **制造文档**：装配说明、BOM（物料清单）、工装要求
- **包装设计**：保护性、传达信息、最小化浪费的包装
- **生产支持**：样品审查、首件检验、质量控制
- **学习**：记录有效做法、改进点、可复用模式

## 理性化预防

| 借口 | 事实 |
|------|------|
| "我先设计形式，再考虑制造" | 制造约束必须从一开始就塑造形式。 |
| "人体工学可以后期在CAD中细化" | 人体工学是设计驱动力，不是细化步骤。早期用模型测试。 |
| "可持续性太贵了" | 可持续设计就是高性价比设计。浪费才是昂贵的。 |
| "用户会去适应产品" | 产品必须适应用户。这才是工业设计的意义。 |

## 危险信号
- 你没有做用户或上下文研究就开始画草图
- 制造可行性没有被考虑
- 人体工学测试被跳过
- 你不知道目标成本
- 可持续性没有被处理

## 验证
- [ ] 用户和上下文研究已完成
- [ ] 产品简报和使用场景已记录
- [ ] 至少探索了2-3个形式方向（至少10+个草图）
- [ ] 已用模型进行人体工学测试
- [ ] 制造可行性已评估
- [ ] 工程文档已准备（CAD、图纸、规格）
- [ ] 可持续性和法规要求已检查

→ 从 **01-intake** 开始，然后按照线性流程加上工业设计专属补充。
