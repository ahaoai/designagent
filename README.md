## What Is DesignAgent?

A complete AI-augmented design workflow system, integrating brand strategy, design research, spatial and experience design, AI generation, design critique, and final delivery into a unified operational framework.

**18 skills across 4 layers:**

### Layer 1: Entry
EXTREMELY-IMPORTANT router that forces design work through the appropriate methodology. Anti-Skip Mechanism blocks 10 common AI rationalizations.

### Layer 2: Control
Mode Lock (Lite / Standard / Deep) prevents AI from downgrading execution depth mid-project. Applicability filters on every skill prevent over-engineering simple tasks.

### Layer 3: Execution
7-step linear pipeline (01 Intake → 07 Learn). Each step has Hard Gates, Verification checklists, Decision Logs (03/04/05), and Conflict Resolution (03/05).

### Layer 4: Governance
Phase-by-Phase Execution Audit with Evidence Binding — every audit verdict must cite Step ID + Section + Artifact Snippet. ✓ PASS / ⚠ CONDITIONAL / ✗ BLOCK. Conflict Audit and Decision Consistency Check prevent post-hoc rationalization.

| Step | Capability | What It Does |
|------|-----------|-------------|
| 01 Intake | Project intake | Gather brief, stakeholders, constraints, success criteria — with Hard Gate |
| 02 Discover | Research & define | User research, competitive analysis, problem definition |
| 03 Strategy | Direction setting | Design positioning, brand strategy, creative direction + Decision Log + Conflict Resolution |
| 04 Generate | Concept generation | Idea exploration, prototyping, AI generation + Decision Log |
| 05 Review | Design review | Structured critique, iteration, optimization + Decision Log + Conflict Resolution |
| 06 Deliver | Production & handoff | Design development, specifications, delivery |
| 07 Learn | Verify & archive | Evidence Binding, execution audit, retrospective, knowledge archive |

### Five Professional Domains

Brand Strategy · UI/UX Design · Architecture Design · Interior Design · Industrial Design

### Cross-Cutting Capabilities

Visual Research · AI Generation · Video Generation · Design Narrative · Design Thinking

## Workflow Mode

DesignAgent runs in **Adaptive Execution Mode**. It scales to the task:

- **Lite** (2-3 steps): "Quick logo" / "Suggest a color palette"
- **Standard** (5-7 steps): Default for design projects
- **Deep** (7+ steps): Full brand identity systems, multi-stakeholder UX redesigns

Mode is locked once selected — cannot downgrade without user approval.

Example — you say: "I want to create a coffee brand":

```
Entry Router → Mode: Standard
01 Intake → 02 Discover → 03 Strategy (Decision Log + Conflict Check)
→ 04 Generate (Decision Log) → 05 Review (Decision Log + Conflict Check)
→ 06 Deliver → 07 Learn (Evidence Audit + Conflict Audit + Decision Consistency Check)
```
